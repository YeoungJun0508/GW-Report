### 개선 사항

- 공룡게임 페이지 로드 후 바로 시작하게 수정함
- 챗봇 초기화 버튼 추가함 
- 사용자 메세지를 즉각 대화메세지에 적용


### 개선해야할 점
- 닮은 연예인 카메라 속도 개선
- 닮은 연예인 입력 데이터 초기화 버튼 추가
- 표정 공룡 게임 노래 추가 및 이미지 수정
- 챗봇 로딩 속도 혹은 로딩 UI 개선
- 챗봇 템플릿 개선
- SST,TTS 모델 적절히 넣기
